# FaaS \(Functions as a Service\) or Serverless

Pros:

* DevOps is not required
* Reduce security footprint
* Isolated
* Pay for what you use

Cons:

* Harder debugging
* Increased complexity on infrastructure

Good for:

* Teams with poor devops

When to use FaaS?

* Unpredictable demands
